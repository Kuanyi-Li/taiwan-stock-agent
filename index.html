🤖 AI 智能選股推薦
Claude AI 根據你的投資目標與現有持股，推薦適合長期投資的標的
如何串接 Claude API：
1. 前往 console.anthropic.com 免費申請帳號
2. 到 API Keys 頁面建立一個新的 API Key（格式：sk-ant-...）
3. 點右上角 ⚙ 設定，在「Anthropic API Key」欄位填入 Key
4. 儲存後回到此頁面，點「AI 分析推薦」即可使用
⚠️ API Key 儲存在本機 localStorage，不會上傳任何伺服器。每次呼叫 Claude API 會從你的帳號扣除用量費用（約 $0.01 USD/次）。
